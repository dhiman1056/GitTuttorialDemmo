git init -- to create new git repo or to initilize git repo (create empty git reposistory)
git add --a (means move all files in staging area)(--a means all)
git commit -m "Initial commit"
git log (to check all commit)

rm -rf .git (to delete all files in git repo)
git diff (to compare working directory and staging area)
git diff --staged (compare previous with current  staging area)
git commit -a -m "message" (to skip staging area)
git rm third.txt (to remove a file)
git mv first.txt first_renamed.txt (to renam a file)
git rm --cacahed filename (to untrack a file)

git log -p
git log -p -3 (will show only 3 commit changes)
git log --stat (short summary for commit message)

how to unstag a file in git?
git restore --staged filename

how to restore or unmodified a file?
git checkout -- filename

Let suppose you have done some changes  and you want to go back to previous commit
then use this command
